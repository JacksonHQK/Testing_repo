---------------------------Existing instances
Attach "myManageInstanceRoleforSSM" IAM role into the existing instance

Create a temporary directory on the instance
mkdir /tmp/ssm

Change to temporary directory
cd /tmp/ssm

Run the following commands to download the SSM installer
curl https://s3.amazonaws.com/ec2-downloads-windows/SSMAgent/latest/linux_amd64/amazon-ssm-agent.rpm -o amazon-ssm-agent.rpm

Then the SSM installers
sudo yum install -y /tmp/ssm/amazon-ssm-agent.rpm

Run the following command to confirm the SSM agent in running. The command should return "amazon-ssm-agent is running."
service amazon-ssm-agent status
